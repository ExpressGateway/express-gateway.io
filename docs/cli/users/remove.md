### Description

Remove one or more users.

### Usage

```shell
eg users remove [options] <user_id|user_name..>
```

### Options

| Name, longform | Type      | Default | Description       |
| ---            | ---       | ---     | ---               |
| `-q, --quiet`  | `boolean` | `false` | Only show user ID |
