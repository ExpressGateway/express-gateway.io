### Description

Update a single user.

### Usage

```shell
eg users update <user_id|user_name> [options]
```

### Options

| Name, longform   | Type      | Default | Description                              |
| ---              | ---       | ---     | ---                                      |
| `-p, --property` | `string`  |         | User property in the form [-p 'foo=bar'] |
| `-q, --quiet`    | `boolean` | `false` | Only show user ID                        |
