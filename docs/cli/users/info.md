### Description

Show details for a single user.

### Usage

```shell
eg users info <user_id|user_name> [options]
```

### Options

| Name, longform | Type      | Default | Description       |
| ---            | ---       | ---     | ---               |
| `-q, --quiet`  | `boolean` | `false` | Only show user ID |
