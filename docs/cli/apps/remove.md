### Description

Remove one or more apps.

### Usage

```shell
eg apps remove [options] <app_id..>
```

### Options

| Name, longform | Type      | Default | Description      |
| ---            | ---       | ---     | ---              |
| `-q, --quiet`  | `boolean` | `false` | Only show app ID |
